  1. 总结

    • 哪些是node.js原生的方法

    • 哪些是express的方法

    • 第三方模块

      1. 搭建项目基本的文件夹和文件结构 views文件夹、assets文件夹、app.js、 npm init -y 初始化一下
      2. 在 app.js 中，从上到下，依次去配置如下这6步：
      • 创建express服务器并启动
      • 配置模板引擎
      • 托管静态资源
      • 配置body-parser
      • 配置express-session
      • 配置路由
      1. 先尝试着，去渲染一个 index 页面出来；
      2. 尝试着，先做一个功能出来，比如，先把 首页上的列表给渲染好；再去完成数据的增删改查（CRDU）c: create r：read u：update d： delete
      3. 比如，要开发注册功能了，先绘制注册的流程图，再设计注册的 API接口文档，然后，根据 流程图 写后台接口，根据 API接口文档 写前端页面调用接口；
